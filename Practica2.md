Ejercicio_1b
============
```Assembly

PA EQU 30h

ORG 1000H
 PRENDIDA DB "Llave prendida"
 APAGADA DB "Llave apagada"
 FIN_APAGADA DB ?

ORG 2000H
 MOV AL, 0FFh
 OUT CA, AL     ;TODO SALIDA
 IN AL, PA
 ; poner en 0 todos los bits menos el m√°s sig

 ; si es 0

 JZ ESTA_APAGADA
 ; esta prendida

 ESTA_APAGADA: MOV BX, OFFSET APAGADA

END
```
